# GeoRaster **WIP**
Very simple geographical raster interaction built on top of [ArchGDAL](https://github.com/yeesian/ArchGDAL.jl/) and [GDAL](https://github.com/JuliaGeo/GDAL.jl).

A GeoArray is a Array of Union{T, Nothing}, an AffineMap for calculating coordinates based on the axes and an crs to interpret these coordinates into in the real world.
